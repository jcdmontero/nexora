# Pruebas de Certificación Funcional Completa

Este directorio ha sido creado para alojar la suite de **Certificación Funcional Completa del ERP**, aislada de los tests E2E y unitarios que corren actualmente, para no interferir con el trabajo del otro agente.

Aquí se alojarán:
- Configuración de Playwright específica para certificación.
- Scripts de simulación de operación comercial masiva.
- Evidencias (capturas, videos, logs).
